רשימת כל הנתיבים של מערכת המסעדה

אפליקציית משתמשים (Users)

Method URL Description
GET/api/users/קבלת רשימת כל המשתמשים
POST/api/users/יצירת משתמש חדש
GET/api/users/<id>/קבלת פרטי משתמש מסוים
PUT/api/users/<id>/עדכון פרטי משתמש
DELETE/api/users/<id>/מחיקת משתמש מסוים

אפליקציית הזמנות מקומות (Reservations)

Method URL Description
GET/api/reservations/tables/קבלת רשימת כל השולחנות
POST/api/reservations/tables/יצירת שולחן חדש (למשתמשים עם הרשאת is_staff בלבד)
GET/api/reservations/reservations/קבלת כל ההזמנות
POST/api/reservations/reservations/create_reservation/יצירת הזמנה חדשה
PUT/api/reservations/reservations/<id>/עדכון הזמנה (למשתמשים עם הרשאת is_staff בלבד)
DELETE/api/reservations/reservations/<id>/ביטול הזמנה (למשתמשים עם הרשאת is_staff בלבד)

אפליקציית תפריט (Menu)

Method URL Description
GET/api/menu/items/קבלת רשימת כל פריטי התפריט
POST/api/menu/items/הוספת פריט חדש לתפריט
PUT/api/menu/items/<id>/עדכון פריט בתפריט
DELETE/api/menu/items/<id>/מחיקת פריט מהתפריט
GET/api/menu/categories/קבלת רשימת כל הקטגוריות
POST/api/menu/categories/הוספת קטגוריה חדשה
PUT/api/menu/categories/<id>/עדכון קטגוריה
DELETE/api/menu/categories/<id>/מחיקת קטגוריה

אפליקציית הזמנות אוכל (Orders)

Method URL Description
GET/api/orders/food/קבלת רשימת כל ההזמנות
POST/api/orders/food/יצירת הזמנה חדשה
GET/api/orders/food/<id>/קבלת פרטי הזמנה מסוימת
PUT/api/orders/food/<id>/עדכון סטטוס או פרטי הזמנה
DELETE/api/orders/food/<id>/ביטול הזמנה
